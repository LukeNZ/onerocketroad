Style Guide

It is important to follow the One Rocket Road styleguide defined here when authoring articles.

Grammar

  • Use hyphens, en-dashes, and em-dashes correctly.
  • Always use superscript to format the date of the month. 18th May, not 18th May.

Markdown

  • Always use markdown semantically. Never use an second-level header if you haven't already used a first-level header.
